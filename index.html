专业技能

WebGIS 开发

专注于构建高性能的地理信息系统，掌握 OpenLayers、Leaflet、ArcGIS API 等主流框架， 具备丰富的空间数据处理和地图可视化经验。

数据可视化

运用 D3.js、ECharts、Three.js 等技术创建交互式数据可视化作品， 将复杂的地理和环境数据转化为直观易懂的图表和动画。

全栈开发

希望精通前端技术栈（HTML5、CSS3、JavaScript、React）和后端开发， 注重用户体验设计、性能优化和无障碍访问。
