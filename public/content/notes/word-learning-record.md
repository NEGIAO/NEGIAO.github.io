## 常用词汇

### GIS 相关

| 英文 | 中文 | 例句 |
|------|------|------|
| spatial analysis | 空间分析 | Spatial analysis is fundamental to GIS. |
| georeferencing | 地理配准 | Georeferencing aligns data to a coordinate system. |
| raster | 栅格 | Raster data represents continuous phenomena. |
| vector | 矢量 | Vector data consists of points, lines, and polygons. |
| buffer | 缓冲区 | Create a 1km buffer around the river. |
| overlay | 叠加 | Overlay analysis combines multiple layers. |
| interpolation | 插值 | Use kriging for spatial interpolation. |
| topology | 拓扑 | Topology defines spatial relationships. |

### 编程相关

| 英文 | 中文 | 例句 |
|------|------|------|
| iterate | 迭代 | Iterate through the list of features. |
| traverse | 遍历 | Traverse the binary tree in order. |
| concatenate | 拼接 | Concatenate the two strings. |
| instantiate | 实例化 | Instantiate a new object. |
| deprecate | 弃用 | This method is deprecated. |
| refactor | 重构 | Refactor the code for better readability. |

### 学术写作

| 表达 | 用途 | 例句 |
|------|------|------|
| This study aims to... | 研究目的 | This study aims to investigate... |
| The results indicate that... | 结果陈述 | The results indicate that the method is effective. |
| In contrast to... | 对比 | In contrast to previous studies... |
| Furthermore,... | 递进 | Furthermore, we found that... |
| To sum up,... | 总结 | To sum up, the proposed method... |
